# AI编辑助手 — 给朋友用的安装说明

> 插件已部署在公网，朋友**不需要**安装 Node.js，**不需要**在 Word 里找「上传我的加载项」（很多中文版 Word 没有这项）。

---

## 环境要求

- Windows 10 / 11（Mac 见文末备用方式）
- Microsoft Word 2016 或更高（推荐 Microsoft 365 桌面版）
- 能正常上网

---

## 第一步：安装插件（推荐，约 1 分钟）

### 方式 A：一键脚本（最简单）

1. 向朋友发送压缩包，内含：
   - `install-friend.ps1`（安装脚本）
2. 朋友**右键** `install-friend.ps1` → **使用 PowerShell 运行**
   - 若提示无法运行，在 PowerShell 中执行：
     ```powershell
     powershell -ExecutionPolicy Bypass -File "脚本所在路径\install-friend.ps1"
     ```
3. 看到「安装完成」后，**完全退出 Word**（任务管理器确认没有 `WINWORD.EXE`）
4. 重新打开 Word

### 方式 B：手动执行（没有脚本文件时）

在 PowerShell 中**整段复制执行**：

```powershell
$dir = Join-Path $env:USERPROFILE "Documents\AI编辑助手"
New-Item -ItemType Directory -Force -Path $dir | Out-Null
Invoke-WebRequest -Uri "https://milei.dpdns.org/manifest.xml" -OutFile (Join-Path $dir "manifest.xml") -UseBasicParsing
$manifest = Join-Path $dir "manifest.xml"
if (-not (Test-Path "HKCU:\Software\Microsoft\Office\16.0\WEF\Developer")) {
  New-Item -Path "HKCU:\Software\Microsoft\Office\16.0\WEF\Developer" -Force | Out-Null
}
Set-ItemProperty -Path "HKCU:\Software\Microsoft\Office\16.0\WEF\Developer" -Name "a1b2c3d4-e5f6-7890-abcd-ef1234567890" -Value $manifest
Write-Host "安装完成，请重启 Word"
```

然后完全退出 Word，重新打开。

---

## 第二步：打开插件

**正确入口（重要）：**

```
Word 顶部「开始」选项卡 → 右侧「AI编辑助手」按钮
```

**不是**「开始 → 加载项」里的 Office 商店列表（GPT for Word、DeepL 等）。

若「开始」里没有按钮，试：

- **插入** → **我的加载项** → **开发人员加载项** → 点「AI编辑助手」

---

## 第三步：配置（只需一次）

侧栏右上角 **设置**：

| 配置项 | 填什么 |
|--------|--------|
| **代理访问控制** → 代理访问令牌 | 安装者私发给你的令牌 |
| **官方 API** → DeepSeek API Key | 朋友自己在 [platform.deepseek.com](https://platform.deepseek.com) 申请 |

填完后点 **保存设置** → **测试连接** 显示成功即可。

> API Key 和令牌只保存在朋友本机，不会上传到 GitHub。

## 使用

- 选中 Word 文字 → 侧栏自动识别 → 输入指令
- 侧栏按 `Ctrl + F5` 可强制刷新
- 代码更新后朋友**不用重装**，刷新侧栏即可

## 常见问题

### Q：Word 里没有「上传我的加载项」？

正常。很多中文版 Word 没有这项，请用上面的 **一键脚本** 安装，不要走「插入 → 上传」。

### Q：「开始」里没有 AI编辑助手？

1. 确认已执行安装脚本且**完全重启过 Word**
2. 检查文件是否存在：`文档\AI编辑助手\manifest.xml`
3. 重新运行安装脚本

### Q：提示「无效的代理访问令牌」？

Word 里的令牌必须与安装者提供的一致，填好后点 **保存设置**。

### Q：Mac 用户？

需安装 Node.js 后执行：

```bash
curl -o ~/Downloads/manifest.xml https://milei.dpdns.org/manifest.xml
npx office-addin-dev-settings register ~/Downloads/manifest.xml
```
