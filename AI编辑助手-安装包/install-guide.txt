  AI编辑助手 — Word 插件 安装说明
═══════════════════════════════════════

【环境要求】
  · Windows 10 / 11
  · Microsoft Word 2016 或更高（推荐 Microsoft 365）
  · 能正常上网

───────────────────────────────────────
第一步：安装插件（约 1 分钟）
───────────────────────────────────────

  双击运行：安装.bat

  窗口显示 Done 后：
  1. 完全关闭 Word（任务管理器确认无 WINWORD.EXE）
  2. 重新打开 Word

───────────────────────────────────────
第二步：打开插件
───────────────────────────────────────

  Word 顶部「开始」选项卡 → 右侧 → 加载项 → 开发人员加载项 → AI编辑助手

───────────────────────────────────────
第三步：配置（只需一次）
───────────────────────────────────────

  侧栏右上角「设置」：

  1. 代理访问控制 → 填「安装令牌.txt」里的代理访问令牌
  2. 官方 API → DeepSeek → 填你自己的 API Key
     申请地址：https://platform.deepseek.com
  3. 点「保存设置」→「测试连接」显示成功即可

───────────────────────────────────────
安装位置
───────────────────────────────────────

  插件文件保存在：
  文档\AI-Word-Editor\manifest.xml

───────────────────────────────────────
常见问题
───────────────────────────────────────

  Q：安装窗口乱码？
  A：不影响安装。看是否显示 Done，然后重启 Word 即可。

  Q：「开始」里没有按钮？
  A：确认已运行安装.bat 并完全重启 Word，再运行一次。

  Q：提示「无效的代理访问令牌」？
  A：检查「安装令牌.txt」是否完整复制到设置里，并点保存。
